系统终极优化 - Windows 7服务手动优化手册

2009-10-10 16:40:47    编辑：Windows7之家 - Mary Jane     字体：【 大 中 小】

Win7之家（ www.win7china.com）：系统终极优化 - Windows 7服务手动优化手册

 自从XP时代以来，“优化”这个词在国内用户中可以说是耳熟能详了，也正由于此，我们开发出了Vista优化大师（点击下载）、Windows 7优化大师（点击下载）以及这两者的进化版本 -魔方（点此下载）。但对于优化爱好者而言，一些手动的优化措施同样也是必不可少的。在所有优化操作中，最难处理的也就是系统服务的优化了。由于服务种类繁多，与系统内核又息息相关，一不注意就很容易出错，因此，我们在优化大师和魔方中仅提供了基础的服务优化。

那么，对于热衷于优化的老鸟而言，如果想要让自己的系统速度更上一层楼，可以参照如下的表格来进行优化。但是在优化之前，我们有如下建议：

1. 使用Vista一键还原(完美支持Windows7)备份系统

2. 使用优化大师或魔方 - 系统优化 - 服务优化来备份整个服务方案

3. 千万不用使用msconfig来禁用服务，而应该运行Services.msc或打开优化大师和魔方

4. 不建议新手使用

5. 如果忘记备份或没有信心，可以参照“安全优化”中的项目进行设置，这样Windows 7在需要时可以自动启动服务，不会造成致命后果
